Проект состоит из нескольких независимых модулей
1) Создание объектов в БД. Пользователи, локации, камеры, фотографии и прочее
2) Аналитика. Вытаскивание объектов из БД и построение графиков
3) Обработка видео
4) UI на DJango. Предоставляет интерфейс для пунктов 1,2
